But how does it work exactly?

Our software is best suited for people who want to host their personal/hobby/small business website on a custom domain without having to rely on expensive hosting services.


Web2Hub automates the process of downloading the html files from any personal or online server and uploading them to your own github repository. It recursively downloads each sub page and makes sure to preserve the original nesting and structure. So all internal links will still work.


Our tool remembers all your credentials, so you can update the site on GitHub with a single click after making changes in the site editor of your choice. You can easily add custom CSS to hide certain elements on the page.


Web2Hub works perfect for static websites. You can't use the standard form submissions from website builder, but tools like GetForms work as well as all CSS styling, javascript interactions and media i.e. video.


FAQ

What is the tool intended for?

You have your website on the staging domain, but want to push it to a custom domain as quickly as possible. Web2Hub is your one-click solution.

Also Web2Hub is an easy solution to connect your private website (for example hosted on your private server) with your domain, without having to rely on static IPs.
